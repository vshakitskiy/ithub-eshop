## Environment
- Setup Mysql database with docker-compose:
```bash
$> docker-compose up
```
- Run queries from `core/eshop.sql` (TABLES, PROCEDURES, SETUP sections)
- Start php server
```bash
$> sudo php -S localhost:80 -t .
```
- Default admin login details:
`login: root; password: 12345`
